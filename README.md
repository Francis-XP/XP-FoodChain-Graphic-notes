## 🚀 快速启动

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python3 app.py
```

启动后浏览器访问 **http://localhost:5188** 即可使用

## ✨ 功能一览

### 📝 图文笔记生成
输入产品名称、核心卖点、目标受众等信息，自动生成多页小红书风格图文笔记

- 内置叙事结构：**封面吸引 → 误区揭示 → 解决方案 → 科学背书 → 行动号召**
- 支持 **1 / 3 / 5** 张按需生成
- 竖版 **3:4** 比例，适配移动端浏览

### 🖼️ 参考图上传
- 上传产品图（支持多张），生成时按页轮换
- 上传 IP 形象图，引导画面风格

### 🎨 内容主题

| 主题 | 说明 |
|:---:|------|
| 0添加剂科普 | 揭示「0添加≠0添加剂」认知误区 |
| 天然食材 | 天然食材替代人工添加剂方案 |
| A2绵羊乳 | A2型绵羊乳的健康价值科普 |
| 产品介绍 | 产品卖点与功能展示 |

### 💅 视觉风格

| 风格 | 特点 |
|:---:|------|
| IP统一版 | 品牌IP主导，课堂互动风 |
| 温馨质感版 | 暖色调，奶油金自然质感 |
| 现代冲击版 | 暗色背景，霓虹高对比 |

### 📦 其他功能
- 📜 生成历史记录，支持查看和删除
- 📥 一键打包 **ZIP** 下载全部图片
