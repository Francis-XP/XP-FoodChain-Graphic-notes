# XP-FoodChain 图文笔记生成器

AI 驱动的小红书图文笔记生成工具，支持自定义产品信息、上传参考图、多风格多主题一键生成。

---

## 快速启动

```bash
pip install -r requirements.txt
python3 app.py
```

浏览器访问：**http://localhost:5188**

---

## 功能概览

**图文笔记生成**
- 输入产品名称、核心卖点、目标受众等信息，自动生成多页小红书风格图文笔记
- 内置叙事结构：封面吸引 → 误区揭示 → 解决方案 → 科学背书 → 行动号召
- 支持 1 / 3 / 5 张按需生成，竖版 3:4 适配移动端

**参考图上传**
- 支持上传产品图（多张），生成时按页轮换使用
- 支持上传 IP 形象图，引导画面风格

**内容主题**
- 0添加剂认知误区科普
- 天然食材替代方案
- A2绵羊乳健康科普
- 产品介绍

**视觉风格**
- IP统一版：品牌IP主导，课堂风
- 温馨质感版：暖色调，自然质感
- 现代冲击版：暗色背景，霓虹高对比

**其他**
- 生成历史记录，支持查看和删除
- 一键打包 ZIP 下载全部图片
