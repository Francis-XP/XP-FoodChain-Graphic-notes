# XP-FoodChain 图文笔记生成器

AI 驱动的小红书图文笔记生成工具，支持自定义产品信息、上传参考图、多风格多主题一键生成。

---

## 快速启动

```bash
pip install -r requirements.txt
python3 app.py
```

浏览器访问：**http://localhost:5188**

---

## 功能概览

- **AI 生成**：输入产品信息，自动生成多页图文笔记图片
- **参考图上传**：支持上传产品图、IP形象图引导生成风格
- **多主题**：0添加剂科普 / 天然食材 / A2绵羊乳 / 产品介绍
- **多风格**：IP统一版 / 温馨质感版 / 现代冲击版
- **历史记录**：查看和管理历史生成记录
- **一键下载**：打包下载生成的全部图片

---

## 技术栈

Python 3 + Flask | HTML5 + CSS3 + JavaScript
